# Security Policy

## Supported Versions

Derzeit wird die `main` Branch-Version aktiv gepflegt.

## Reporting a Vulnerability

Bitte melde Sicherheitsprobleme verantwortungsvoll und nicht oeffentlich.

Sende eine vertrauliche Meldung an den Repository-Owner mit:

- Kurzbeschreibung der Luecke
- Reproduktionsschritten
- moeglichem Impact
- ggf. Vorschlag zur Behebung

Wir bestaetigen den Eingang zeitnah und halten dich zum Status auf dem Laufenden.
